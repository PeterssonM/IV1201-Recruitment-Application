Website description

The following text provides information about Tivoli, a web application designed for the purpose of facilitating the recruitment process for a company.

This website was developed by Daniel Mebrahtu Redi and Maximilian Petersson as part of their IV1201 VT23 'Architecture and design of global applications' project.

The Tivoli web application is intended to serve as a platform for job applicants to apply for positions within the company and for recruiters to manage the applications.

To use the application, users are required to log in using their Tivoli account. In case a user does not have a Tivoli account, they can easily register as a new user by clicking on the appropriate button.
